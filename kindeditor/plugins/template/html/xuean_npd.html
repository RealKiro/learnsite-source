学生活动
活动一：了解互联网发展过程中的创新点

查阅资料，了解互联网在不同发展阶段的新特征，并填写“互联网发展大事记”时间轴。

  1. 了解互联网发展的四个阶段：诞生阶段、形成阶段、互联阶段、高速网络阶段
  2. 查阅网络资料，了解每个阶段的代表性创新技术
  3. 在表格中填写对应年份的重大事件
  4. 与同学交流各阶段的新特征，总结创新对互联网发展的推动作用
活动二：我国互联网发展大事记

搜索并整理我国互联网发展历程中的重要事件，选择三个关键点并说明理由。

  1. 回顾我国互联网发展的两个关键年份：
    • 1987年：建成第一个国际互联网电子邮件节点
    • 1994年：全功能接入国际互联网
  2. 通过网络搜索或查阅资料，了解1995年至今的重要互联网事件
  3. 选择你认为最重要的三个事件，填入时间轴
  4. 写下选择理由，并与小组同学分享交流
活动三：“互联网+”生活体验分享

列举三个让你感受到生活品质提升的“互联网+”例子，描述其带来的便利。

  1. 思考日常生活中哪些场景用到了“互联网+”
  2. 列举三个具体例子（如：在线购票、扫码支付等）
  3. 描述每个例子如何提升生活品质（如：节省时间、提供便利等）
  4. 尝试分析这些应用背后的技术支持（如：北斗定位、移动支付等）
  5. 在班级内进行简短分享，听取他人的例子，丰富自己的认知

学习内容详解

一、互联网的创新发展阶段

1. 四个发展阶段

  • 诞生阶段：面向终端的联机系统，多终端共享一台计算机
  • 形成阶段：计算机之间互联，阿帕网诞生
  • 互联阶段：协议标准化，TCP/IP成为核心协议
  • 高速网络阶段：“信息高速公路”普及，应用深入社会各领域

2. 网状结构的优势

节点间有多条通信线路，单点故障不影响整体通信，网络更健壮。

二、我国互联网建设与创新成就

1. 从网络大国到网络强国

  • 网民规模全球第一
  • “互联网+”成果显著：网络零售、数字教育、智能制造等
  • 5G技术领跑全球：建成最大5G网络，应用融入各行业大类
  • 北斗系统全面应用：服务交通、农业、救灾、公共安全等领域

2. 关键概念理解

  • “互联网+”：传统行业借助互联网转型升级
  • “灯塔工厂”：代表智能制造最高水平的先进工厂
  • 5G特点：高速、低时延、大容量
  • 北斗系统：我国自主建设的全球卫星导航系统
